Detailed Description

This Wrapper is the superclass for the different data sets. The class to creating a subclass of this is only used to store the current SerialResponseContainer and its type. The member functions of this Wrapper are called by Fluke189ResponseAnalyser::operator[](unsigned int) This class is a superclass so it can not be constructed.
