Detailed Description

This Wrapper holds the functions for the different data sets. The class to this function is only used to store the current SerialResponseContainer and its type. The member functions of this Wrapper are called by Fluke189ResponseAnalyser. This class can only be constructed by Fluke189ResponseAnalyser because its constructor is private.
